# MacOS Portfolio

A personal web portfolio built with React. Currently, the project features a fully functional, animated macOS-styled Dock, with more desktop environment features actively in development.

## Features

*   **macOS Dock:** Animated and interactive bottom dock mimicking the macOS interface.
*   *(More features like an Interactive Desktop, Functional Apps, and Status Bar are currently in development)*

## Tech Stack

*   **Framework:** React.js
*   **Styling:** CSS3 / SCSS
*   **State Management:** React Context API
